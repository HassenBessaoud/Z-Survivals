# Z Survivals 🎮

Z Survivals is a 2D survival game inspired by the movie *World War Z*, developed as part of an academic SDL/Game Development project.

## 📌 Project Overview

The objective of the game is to survive through different challenges while avoiding obstacles and interacting with gameplay elements in a post-apocalyptic environment.

The project focuses on:
- 2D game development using SDL
- Collision and movement systems
- Interactive gameplay mechanics
- Quiz and challenge integration
- UI and environment design

---

## 🛠️ Technologies Used

- C
- SDL2
- SDL_image
- SDL_ttf
- SDL_mixer
